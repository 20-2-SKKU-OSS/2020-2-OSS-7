📌주요 task 정리
1. html 코드 분석, 기존 코드 분석
2. 크롤러 기능 추가
3. 크롤링된 결과를 통계내고 활용하는 페이지 제작

1. html 코드 분석, 기존 코드 분석 - 윤민철, 이서윤
2. 크롤러 기능 추가 - 정병민
3. 크롤링된 결과를 통계내고 활용하는 페이지 제작 - 서의태, 박다영, 정지혜
